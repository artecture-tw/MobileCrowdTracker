資訊收集與使用

當您下載並使用本應用程式時，應用程式會收集資訊。這些資訊可能包括：

  • 您裝置的網際網路通訊協定位址（例如 IP 位址）

本應用程式不會收集您行動裝置位置的精確資訊。

服務提供者可能會使用您提供的資訊，不時與您聯繫，以向您提供重要資訊、必要的通知和行銷促銷活動。

僅有匯總的、匿名的資料會定期傳輸到外部服務，以協助服務提供者改進本應用程式及其服務。 服務提供者可能會以本隱私權聲明中描述的方式與第三方分享您的資訊。

服務提供者可能會揭露使用者提供和自動收集的資訊：

  • 根據法律要求，例如遵守傳票或類似的法律程序；
  • 當他們真誠地相信揭露對於保護他們的權利、保護您或他人的安全、調查詐欺或回應政府請求是必要的；
  • 與他們信任的服務提供者，這些服務提供者代表他們工作，不會獨立使用我們向他們揭露的資訊，並已同意遵守本隱私權聲明中規定的規則。
